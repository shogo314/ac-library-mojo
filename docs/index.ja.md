# ac-library-mojo Docs

## 📦 概要

**ac-library-mojo** は AtCoder の [ac-library](https://github.com/atcoder/ac-library) を Mojo 言語に移植したもので、競技プログラミング向けに設計された高性能なアルゴリズムライブラリです。
